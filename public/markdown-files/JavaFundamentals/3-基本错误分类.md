# 基本错误分类

- 编译时错误：程序语法错误导致的异常

  ```
  int x = "hello";  // java: 不兼容的类型: java.lang.String无法转换为int
  ```

- 运行时错误：空指针、数组越界等

  ```java
  public class Test {
  //    Exception in thread "main" java.lang.NullPointerException: Cannot invoke "String.length()" because "str" is null
  //          异常发生的异常                       异常的分类                      异常发生原因
  //    at Test.main(Test.java:4)
  //          异常发生位置
      public static void main(String[] args) {
          String str = null;
          System.out.println("Hello World" + str.length());
      }
  }
  ```
